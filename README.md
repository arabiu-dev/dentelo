### Run Locally

To run **Dentelo** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/arabiu-dev/dentelo.git
```

Windows:

```bash
git clone https://github.com/arabiu-dev/dentelo.git
```
